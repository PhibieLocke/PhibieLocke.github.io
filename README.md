# i-love-eva-elfie

First version of an SEO-focused Eva Elfie fan page with a TikTok-inspired layout, gamified fan engagement, and a GitHub Pages deployment workflow.

## Launch on GitHub Pages

1. Push this repository to GitHub.
2. In **Settings → Pages**, set **Source** to **GitHub Actions**.
3. Push to `main` to trigger deployment via `.github/workflows/deploy-pages.yml`.

## SEO setup checklist

- Replace all `https://YOUR-USERNAME.github.io/i-love-eva-elfie/` placeholders in:
  - `index.html`
  - `robots.txt`
  - `sitemap.xml`
- Add a `preview.png` image at the repository root for social sharing cards.
